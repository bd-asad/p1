How to change Images? To change any image you want to change, just go to image folder and select that image. Copy its name and then delete. Now keep any image in here and paste the copied name. Image name should be same.
For better result please keet the image size same.

How to change text? To change any text you wish. First you need to select the page from where you want to change. To see the page name just see the browser address. and there will be pages on the last. such as ".../index.html".
Now go to the template folder select that page such as index.html and Open this with any text editor such as notepad+, sublimetext etc. Now you need to find that text to change. Just find and change.

How to change links? As like text changing method. You need to find that link on any text editor. Now there will be a syntex " href="www.link.com" " Now you can change the link address here.

How to add more images/content? To add more image or content you need to go on html page as like text editor need to copy a specific content you want to increase and then paste it on bellow line. Then change the image name and keep the same image on the same forder.
